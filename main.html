03

TX HASH → VALUE 0–999

Your transaction hash is processed through a single modulo operation producing a perfectly uniform value between 0 and 999. No bias. No dead zones. No manipulation.

// Core input function — must not be modified
uint16 value = uint16(uint256(txHash) % 1000);
04

STORED IN THE ROLLING ARRAY

Written into the next slot in the 52-entry ring buffer. The write head advances with every qualifying transaction. When slot 52 is reached it wraps to slot 0. Your entry is live until overwritten.

COLLISION DETECTION

When your transaction lands, its value is compared against all 8 Fibonacci-index slots. If it exactly matches an active slot — that is a collision. Both wallets paid immediately and atomically.

06

DUAL PAYOUT — ATOMIC

0.52% of the current prize pool split 50/50 between the incoming sender and the matched slot’s sender. Both transfers execute in the same transaction — if either fails, both revert. No partial payouts ever.

07

ASYMPTOTIC DECAY — POOL NEVER EMPTIES

Each win pays 0.52% of the current balance, not a fixed amount. The pool can never reach zero — payouts shrink as balance shrinks. Buy tax (2.718%) continuously replenishes it. Self-sustaining in perpetuity.

1. OVERVIEW

Project 52F is a deterministic DeFi token on QF Network (Substrate-based, SPIN consensus, RISC-V, 0.1s block finality). Total supply equals the first 11 digits of 52 factorial — a fixed, immutable constant. Every parameter is anchored to a mathematical constant or derived from 52, the protocol identity.

The core innovation is the Birthday Paradox Engine: an on-chain collision-detection reward system where two independent transactions whose processed hashes match trigger a dual simultaneous payout. No randomness oracle. No admin key for prize distribution. Pure deterministic logic.

The following constants are immutable and must be hardcoded exactly as specified.

// Fibonacci indices [1,2,3,5,8,13,21,34] as O(1) bitmask
uint256 constant FIB_MASK = 0x40020212E; // 17,181,974,830 decimal

struct ArraySlot {
  uint16 value; // 0–999 modulo result

// Hash processing — must not be modified
