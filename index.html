The Birthday Paradox Engine

Unlike traditional lotteries, the Paradox is deterministic probability encoded in a sparse matrix. You don't "move" through slots. Each transaction is an independent event.

Critical: This is not a progressive game. You do not advance from slot 1 to 2 to 3. Each transaction hashes to a random slot 0-999. If that slot is a Fibonacci number and occupied, you win. If not, you simply occupy (or steal) the seat with no payout.
