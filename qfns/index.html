About QFNS

The naming service for the QF Network ecosystem

QFNS (QF Naming Service) provides human-readable names for QF Network wallet addresses. Instead of sharing long hexadecimal strings, users can register simple names like "alice" or "bob" that resolve directly to their wallets.

Built on ink! smart contracts with commit-reveal registration, QFNS ensures fair name allocation while preventing front-running attacks. Every transaction burns 5% of fees, creating deflationary pressure on the QF token.
